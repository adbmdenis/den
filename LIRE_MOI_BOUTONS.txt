                    SOLUTION: BOUTONS ADMIN PANEL NE FONCTIONNENT PAS
═══════════════════════════════════════════════════════════════════════════════

PROBLÈME:
  Les boutons du panel admin affichent l'erreur:
  "Uncaught ReferenceError: showModal is not defined"

CAUSE:
  Le serveur utilise l'ANCIEN fichier admin_panel.py en cache.
  Python garde les modules en mémoire et ne recharge pas automatiquement.

SOLUTION RAPIDE (3 étapes):
═══════════════════════════════════════════════════════════════════════════════

  1. ARRÊTER LE SERVEUR
     ► Appuyez sur Ctrl+C dans le terminal où le serveur tourne
     ► OU double-cliquez sur: REDEMARRER_MAINTENANT.bat

  2. REDÉMARRER LE SERVEUR
     ► Double-cliquez sur: REDEMARRER_MAINTENANT.bat
     ► OU tapez: python server.py

  3. VIDER LE CACHE DU NAVIGATEUR
     ► Appuyez sur F12 (DevTools)
     ► Clic droit sur le bouton Actualiser
     ► Choisir "Vider le cache et actualiser de manière forcée"
     ► OU: Ctrl+Shift+Delete → Effacer les données → Ctrl+F5

═══════════════════════════════════════════════════════════════════════════════

VÉRIFICATION:
═══════════════════════════════════════════════════════════════════════════════

  Après redémarrage, ouvrez la console du navigateur (F12) et tapez:
  
  > showModal
  
  Résultat attendu:
  ƒ showModal(id) { document.getElementById(id).classList.add("active"); }
  
  Si vous voyez "undefined", le cache n'est pas vidé!
